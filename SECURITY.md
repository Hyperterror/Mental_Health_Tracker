## Reporting a Vulnerability

Use this section to tell people how to report a vulnerability.

Please report any security vulnerabilities via email to `security@mindfulprep.example.com`. Do not report them via public GitHub issues.

We will acknowledge your report within 48 hours and typically release a patch within 1 week for critical vulnerabilities.
